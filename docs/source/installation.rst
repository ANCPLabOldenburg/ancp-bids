Install using `pip`
-------------------
ancpBIDS is available via `PyPi.org <https://pypi.org/project/ancpbids/>`_

.. code-block::
   :caption: Initial installation

       pip install ancpbids

.. code-block::
   :caption: Upgrade existing

       pip install --upgrage ancpbids
